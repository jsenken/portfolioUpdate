Pool party was a website created to allow friends to compete against each other by answering a series of questions before a big sporting or entertainment event. Blood stream was a group project put together during the coding bootcamp. It is a website for video game consumers to go to find the most up to date information about a video game. A user would search a game and twitch streams, reddit board posts and critical reviews would show. I specifically worked on the API calls to IGDB to pull review information.
